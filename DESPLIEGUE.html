💻 Opción 1: Ejecutar Localmente

Método A: Abrir directamente

1. Abre tu gestor de archivos

2. Navega a la carpeta del portafolio

3. Haz doble click en index.html

4. ¡Se abrirá en tu navegador! 🎉

Método B: Servidor local (Recomendado)

Con Python 3:

cd /ruta/del/portafolio python3 -m http.server 8000

Con Python 2:

python -m SimpleHTTPServer 8000

Con Node.js:

npx http-server -p 8000

Luego abre en tu navegador: http://localhost:8000

✓ Ventaja: No necesitas instalar nada, es la forma más fácil

🐙 Opción 2: GitHub Pages (Gratuito)

Paso 1: Crear repositorio en GitHub

  1. Ve a github.com/new
  2. Nombre del repositorio: tu-usuario.github.io
  3. Selecciona "Public"
  4. Click en "Create repository"

Paso 2: Subir archivos

# Desde la carpeta del portafolio git init git add . git commit -m "Initial commit - Portafolio web" git branch -M main git remote add origin https://github.com/tu-usuario/tu-usuario.github.io.git git push -u origin main

Paso 3: Configurar GitHub Pages

  1. Ve a tu repositorio en GitHub
  2. Settings → Pages
  3. Source: Branch "main"
  4. Espera 1-2 minutos
  5. Tu sitio estará en: https://tu-usuario.github.io
✓ Ventaja: Totalmente gratuito, integrado con Git, actualizaciones fáciles

▲ Opción 3: Vercel (Recomendado)

Paso 1: Conectar con GitHub

  1. Ve a vercel.com
  2. Sign up with GitHub
  3. Autoriza Vercel

Paso 2: Importar proyecto

  1. Click "New Project"
  2. Selecciona tu repositorio
  3. Configuración por defecto está bien
  4. Click "Deploy"
✓ Ventaja: Deployment automático, SSL gratis, dominio gratis, rendimiento excelente

🎯 Opción 4: Netlify

Método A: Drag & Drop (Más fácil)

  1. Ve a netlify.com
  2. Haz scroll hasta "Want to deploy a new site?"
  3. Arrastra tu carpeta del portafolio aquí
  4. ¡Listo! Tu sitio estará online en segundos

Método B: Conectar con GitHub

  1. Sign up con GitHub
  2. New site from Git
  3. Selecciona tu repositorio
  4. Deploy
✓ Ventaja: Interfaz intuitiva, dominio libre, SSL incluido, updates automáticos

🌐 Conectar Dominio Propio

Si tienes un dominio personalizado (ejemplo.com), conectarlo es fácil:

Para GitHub Pages:

  1. Ve a tu proveedor de DNS (GoDaddy, Namecheap, etc.)
  2. Busca "DNS Records" o "DNS Settings"
  3. Agrega registro A: 185.199.108.153
  4. En GitHub: Settings → Pages → Custom domain
  5. Ingresa tu dominio

Para Vercel/Netlify:

  1. En tu panel de Vercel/Netlify: Domain settings
  2. Add custom domain
  3. Sigue las instrucciones de DNS
  4. Usualmente se resuelve en 24-48 horas

Para mejorar SEO:

Edita el <head> en index.html:

<meta name="description" content="Portafolio de [Tu Nombre]..."> <meta name="keywords" content="ingeniero sistemas, developer, python, javascript"> <meta name="author" content="Tu Nombre">

Para agregar Google Analytics:

  1. Ve a analytics.google.com
  2. Crea una nueva propiedad
  3. Copia el código de tracking
  4. Pégalo antes del </head> en index.html

Para agregar formulario funcional:

Considera usar servicios como:

  • Formspree.io - Fácil integración, gratuito
  • Basin.io - Simple, sin código backend necesario
  • Netlify Forms - Si usas Netlify
  • Firebase - Más profesional, requiere setup

🆘 Solución de Problemas

El sitio no carga

☑️ Verifica que todos los archivos estén en la misma carpeta
☑️ Revisa la consola del navegador (F12)
☑️ Comprueba tu conexión a internet
☑️ Intenta en otro navegador

Dark mode no funciona

☑️ Limpiar caché del navegador (Ctrl+Shift+Del)
☑️ Verificar que script.js está cargado
☑️ Revisar consola para errores

Formulario no funciona

☑️ Esto es normal - es una demostración sin backend
☑️ Para producción, usa Formspree o Firebase
☑️ Ver sección de "Para agregar formulario funcional"
